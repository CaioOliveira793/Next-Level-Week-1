# Mobile

## Como Executar

No diretorio do mobile, execute

```cmd
yarn install
yarn start
```

ou

```cmd
npm install
npm start
```

### Dispositivo Físico

Para utilizar seu dispositivo instale o app expo na sua loja de aplicativo e escaneie o QR code.

### Emulador

Caso preferir utilizar um emulador, o expo irá detectá e executar o projeto.
